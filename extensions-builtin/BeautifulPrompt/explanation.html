Information
Name Description
Model The selected model. Local models, API, hugging face models can be used. Selecting API or None will free up GPU memory.
API Token Authentication token when using the API, can be left empty when not using the api.
Top K When appending a word to the prompt, pick out of K most likely candidates.
Top P Only the smallest set of most probable tokens with probabilities that add up to top_p or higher are kept for generation.
Temperature When appending a word to the prompt, the greater temperature is, the more chance to pick an unlikely candidate.
Repetition penalty The greater the value is, the less likely repeated tearms are to appear in prompt. 1.0 means no penalty.
Max length The total maximum length instruction, raw prompt and generated prompt in tokens.
Num prompts The number of prompts to generate.
Negative prompt Also generate a pre-defined negative prompt.
